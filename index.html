Easy to Use

Skyle was designed to improve React Native's styling and animation workflow.

Performant

Uses React Native's StyleSheet and Animated to achieve near-native performance.

Customizable

Includes many ways to customize your styling experience (Theming, Custom Preprocessors and more!)

Clean Design

Skyle's design is very easy to write and understand. It is based off of React Native's StyleSheet and CSS.

CSS-Inspired Features

Adds many popular CSS features that are missing from standard React Native.

Universal

Skyle can be used with any React Native platform: iOS, Android, Web, Windows, MacOS, and more!
