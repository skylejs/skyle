Easing

Skyle's Easing is an extended version of React-Native's Easing.

@see https://reactnative.dev/docs/easing

import { Easing } from 'skyle';

As it extends React Native's Easing, all standard properties and methods are available.

Standard CSS easings#

@see https://developer.mozilla.org/en-US/docs/Web/CSS/easing-function#syntax

  • ease

  • easeInOut
