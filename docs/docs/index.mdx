## Highlights

- 💡 Beautiful & Easy Syntax
- ⚡ Performant
- 🎨 Highly Customizable
- 🔼 **Expo** Compatible
- 🧩 Works on all Platforms! (**iOS**, **Android**, **Web**, **Windows**, **Mac**, etc.)
- 🌟 Supports **Media Queries**, **Theming**, **Transitions** and more!
- ☑️ Written in TypeScript
