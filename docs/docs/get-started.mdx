## Enabling Decorators

### TypeScript

Enable the compiler option `"experimentalDecorators": true` in your _tsconfig.json_.

### Babel 7

Install support for decorators:

<Tabs
  defaultValue='yarn'
  values={[
    { label: 'npm', value: 'npm' },
    { label: 'Yarn', value: 'yarn' },
  ]}>
  <TabItem value='npm'>

```bash
npm install --save-dev @babel/plugin-proposal-class-properties @babel/plugin-proposal-decorators
```

  </TabItem>
  <TabItem value='yarn'>

```bash
yarn add -D @babel/plugin-proposal-class-properties @babel/plugin-proposal-decorators
```

  </TabItem>
</Tabs>

And enable it in your _babel.config.js_ file (note that the order is important):

```js
{
  "plugins": [
    ["@babel/plugin-proposal-decorators", { "legacy": true }],
    ["@babel/plugin-proposal-class-properties", { "loose": false }]
  ]
}
```
