The default export includes many methods for customizing Skyle's internals.

```ts
import Skyle from 'skyle';
```

## Methods

### configure

```ts
function configure(options?: object): void;
```

### addAlias

```ts
function addAlias(aliases: object): void;
```

### removeAlias

```ts
function removeAlias(aliases: object): void;
```

### setPreprocessor <span class="badge change">Experimental</span>

```ts
function setPreprocessor(preprocessors: object): void;
```
