Length Units

@see https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/Values_and_units

// Examples
width: '100vw',
fontSize: '10em',
transform: [{ scale: '10px' }],
borderRadius: '100vh',
info

By default (if no unit is specified), 'px' will be used.

e.g.: 5 = 5px
