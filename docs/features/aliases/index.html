Aliases

You can easily add/remove aliases using the addAlias and removeAlias functions.

Skyle.addAlias({
property1: 'alias1',
// Use array for multiple
property2: ['alias2', 'alias3'],
...
});
Skyle.removeAlias({
property1: 'alias1',
...
});
// Example
Skyle.addAlias({
color: 'textColor'
});
/// Use in StyleSheet
textColor: 'red',
